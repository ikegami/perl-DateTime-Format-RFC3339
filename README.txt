DateTime-Format-RFC3339

DateTime::Format::RFC3339 parses and formats RFC3339
datetime strings.

INSTALLATION

To install this module, run the following commands:

    perl Makefile.PL
    make
    make test
    make install

DEPENDENCIES

This module requires these other modules and libraries:

    DateTime
    Test::More
    strict
    version
    warnings
